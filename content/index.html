Documentation for the Bytemark BigV (bigv.io) API

Bytemark's new BigV service does not currently have any official documentation.

This will, at some point, be ported over to the bigv.io site and form the official documentation, but currently serves as a way of providing some notes to as an source of reference for the API.

If you have any information, additions or corrections, then please do let me know.

The source code for this site is available here: https://github.com/ichilton/bytemark-bigv-api-doc so feel free to raise an issue or even a pull request. You can also e-mail me at: ian [dot] chilton [at] bytemark [dot] co [dot] uk or tweet me.
