Unofficial Documentation of the Bytemark BigV (bigv.io) API

The API behind Bytemark's BigV service is currently not publically documented.

This site serves as a way of providing some notes to serve as an initial source of reference for the API. Bytemark have supplied some of this information, but some of this is based on looking at the BigV command line client and it's log files.

Many thanks to Tom Molesworth for providing me with his notes which made a great starting point when I was putting this together.

Please note that these notes are unofficial and not guarenteed to be correct. Use at your own risk. I am not responsible for any problems you have, or charges you incur on your account resulting from the use of this information.
